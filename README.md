# BerlinX Bikes
 
 A shiny app to explore bike lane types and bike accidents in Berlin.
 
Created for the CorrelAidX Challenge 2020 “Analysing and visualising German regional statistics with datenguidepy” by Andreas Neumann, Cédric Scherer, Saleh Hamed & Steffen Reinhold as contribution by CorrelAidX Berlin.

### Run the app

```
devtools::install_github("Z3tt/berlinxbikes")
berlinxbikes::run_app()
```
